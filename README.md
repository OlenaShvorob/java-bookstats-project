BookStats

Консольна програма на Java для:

читання всіх JSON-файлів із src/main/resources/data

підрахунку статистики за атрибутами: author, genre, year_published

збереження статистики у форматі XML в src/main/resources/output

порівняння швидкодії під час парсингу в 1, 2, 4 та 8 потоків

Запуск

Відкрити проєкт у IntelliJ IDEA.

Запустити клас Main:
src/main/java/ua/olena/bookstats/cli/Main.java

Необов'язково: можна передати аргументи командного рядка

перший аргумент — шлях до папки з JSON

другий аргумент — атрибут (author, genre, year_published)
Без аргументів програма використовує значення за замовчуванням.

Результати

XML-файли зі статистикою зберігаються в:
src/main/resources/output

Статистика (за тестовими JSON-файлами)

кількість книг: 9

унікальних авторів: 9

унікальних жанрів: 10

Швидкодія парсингу

(приклад реального запуску)
1 потік: ~95 ms
2 потоки: ~5 ms
4 потоки: ~10 ms
8 потоків: ~0 ms

Збільшення кількості потоків пришвидшує обробку, оскільки JSON-файли незалежні й можуть оброблятися паралельно.

Тести

Юніт-тести знаходяться в:
src/test/java/ua/olena/bookstats/AppTests.java

Тести перевіряють:

коректність читання всіх JSON-файлів

правильність обчислення статистики

відповідність результатів при 1 і 4 потоках

обробку некоректного атрибута
